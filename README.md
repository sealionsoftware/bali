bali
====

A simple imperative JVM dialect with compile time checking of null and concurrent safety.

Comparison with Java:

(out)
- Statics
- Closures
- Implementation inheritance
- Checked exceptions
- Primitives
- Multiple constructors
- Annotations
- Enums
- Protected/Default
- Reflection
- Manual casting

(in)
- Constants
- Nullable types
- Monitor types
- Run statements
- Named parameters
- Record types
- Runtime type parameters
- Flow typing
