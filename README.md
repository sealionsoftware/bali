bali
====

A simple imperative JVM dialect with compile time checking of null and concurrent safety.

Comparison with Java:

(out)
- Statics
- Closures
- Implementation inheritance
- Checked exceptions
- Primitives
- Multiple constructors
- Annotations
- Enums
- Protected/Default
- Reflection
- Manual Casting

(in)
- Constants
- Nullable Types
- Monitor Types
- Run Statements
- Named parameters
- Record types
